# 📄 Simple Academic CV

A clean and minimal academic CV written in LaTeX.

## ✨ Features
- simple academic layout
- clean typography
- easy to edit and adapt
- suitable for research, applications, and personal websites

## 🛠️ Usage
Compile the file with LaTeX:

```bash
pdflatex main.tex
```

## 📌 Notes
This CV is designed to stay simple, readable, and easy to maintain.
